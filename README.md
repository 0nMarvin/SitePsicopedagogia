# Site Psicopedagoga Flaviane

Este site foi desenvolvido como parte de um estágio voluntário não remunerado para a Doutora Flaviane Pelloso Molina Freitas, destinado ao consultório de Psicopedagogia.
O objetivo principal é oferecer uma imagem digital da clínica, funcionando como um espaço de acolhimento e informação, onde pacientes e interessados possam conhecer:

- A proposta de trabalho da clínica
- Os três serviços principais oferecidos
- Informações de contato e localização

Para Acessa a Prévida do Site: https://0nmarvin.github.io/SitePsicopedagogia/
